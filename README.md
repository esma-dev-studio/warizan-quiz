# 🦊 けいさんマスター

子ども(小1〜小4)向けの 計算れんしゅうゲーム。**たし算・ひき算・かけ算・わり算**の4演算対応。
**ビルド不要・インストール不要・オフラインOK** — `index.html` をダブルクリックするだけで遊べます。

## あそびかた

1. `index.html` をブラウザで開く(ダブルクリック)
2. ➕➖✖️➗ から えんざんを えらぶ
3. あそびかたを えらぶ
   - ⚡ **クイズ** … 10問スピード勝負(演算ごとに3〜4レベル、九九は「だん」も選べる、わり算はあまりつきも)
   - ✏️ **ひっ算** … 4演算ぜんぶ対応(全12レベル)。やり方の解説アニメつきで、くり上がり・くり下がり・2だんのかけ算・「たてる かける ひく おろす」を1ステップずつ入力して練習
4. テンキー(またはキーボード)で答えて **OK!**

全15のクイズパターン(1けた＋1けた 〜 2けた×1けた、あまりのあるわり算まで)は、むずかしいほどXP倍率が高く、やればやるほど共通のレベル・称号が上がります。

## スコア計算

- 正解1問 = **100点** + スピードボーナス + コンボボーナス(連続正解×10、上限+100)
- 獲得XP = スコア ÷ 10。レベルアップに必要なXP = `100 + (レベル-1) × 50`

## ファイル構成

```
warizan-quiz/
├── index.html   … 画面構造(5画面のSPA)
├── style.css    … 子ども向けデザイン・全アニメーション
├── data.js      … 問題生成・バッジ定義・ほめ言葉・localStorage保存
├── effects.js   … 紙吹雪(canvas)・効果音(WebAudio合成、音声ファイル不要)
└── app.js       … ゲーム進行・採点・レベル・ランキングのロジック
```

- 記録は ブラウザの `localStorage`(キー: `warizan-master-v1`)に保存。サーバー不要
- 効果音はホーム画面の 🔊 ボタンでON/OFF

## カスタマイズのヒント

- **難易度調整**: `data.js` の `generateQuestion()` の数値範囲を変更
- **バッジ追加**: `data.js` の `BADGES` 配列に `{id, emoji, name, desc, check}` を追加するだけ(画面は自動で対応)
- **ほめ言葉の追加**: `data.js` の `PRAISES` / `ENCOURAGES` 配列に追記
